# Definition of Done

Work is Done only when:

- every acceptance criterion is evidenced;
- behavior is tested at the appropriate unit, integration, security, and end-to-end levels;
- lint, typecheck, test, and build commands pass when defined;
- documentation, architecture status, risks, and submission evidence are current;
- secret and dependency checks pass;
- the pull request is reviewed as required and merged;
- GitHub Project status and completion evidence are synchronized.
