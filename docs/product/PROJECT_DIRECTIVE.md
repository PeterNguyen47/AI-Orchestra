# Project Directive

**Version:** 1.0 — Frozen July 13, 2026

**Change control:** Any scope change requires a recorded decision in the scope-change log.

## Mission

Build a user-friendly, low-code, governance-first AI architecture composer that helps technical and nontechnical users design, configure, validate, test, understand, monitor, and export governed AI systems.

## Primary user

A solution architect, developer, or technically capable product owner who needs to design a governed AI system without manually assembling every architecture artifact.

## Build Week direction

The primary track is Developer Tools. The MVP is one executable Enterprise RAG question-and-answer architecture with visible guardrails, evaluation, diagnostics, and exports. A CRM or relational-database node is included as a realistic, explicitly simulated component.

## Success boundary

Deliver a coherent, judge-runnable vertical slice before visual polish. Every node is labeled `executable`, `simulated`, or `roadmap`. No claim may exceed observed behavior.

This Version 1.0 adds no requirements beyond the approved package. The authoritative detail is in [MVP_SCOPE.md](MVP_SCOPE.md).
