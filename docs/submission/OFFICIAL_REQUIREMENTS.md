## Operational requirements

- Submission window: July 13, 2026 at 9:00 AM Pacific through July 21, 2026 at 5:00 PM Pacific (8:00 PM Eastern).
- Build a working project using Codex with GPT-5.6 in an eligible track; AI Orchestra selects Developer Tools.
- Submit a category, project description, and a public YouTube demo under three minutes showing the project working, with audio covering Codex and GPT-5.6 use.
- Provide a repository URL for judging/testing. A public repository must use relevant licensing; include setup instructions, sample data when needed, and clear run guidance.
- Highlight how Codex accelerated work, key decisions, and how GPT-5.6 and Codex were used.
- Provide the primary Codex `/feedback` Session ID for the session where most core functionality was built.
- Developer tools must include installation instructions, supported platforms, and a judge path that does not require rebuilding from scratch, such as a demo, sandbox, or test account.
- Request free Codex credits from the Devpost Resources page by July 17, 2026 at 12:00 PM Pacific (3:00 PM Eastern), if desired and available.
